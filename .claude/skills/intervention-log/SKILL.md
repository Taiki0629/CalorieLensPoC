サイクルの記録工程として、介入ログを1件追記する。入力（課題キー・メモ）は引数で渡される。

## 手順
1. `10_management/templates/介入記録テンプレート.md` の項目に沿う。
2. `10_management/介入ログ.md` が無ければテンプレ見出しで新規作成し、該当課題のエントリを追記する。
3. 記入項目: 任せた範囲 / 採用率の体感 / 人間の介入（何を・なぜ） / AIレビューで検出した指摘 / ハマり・学び。
4. このセッションでの実際のやり取りを踏まえ、誇張せず事実ベースで書く。

これは記事②「AI駆動開発でどこまで任せられたか」の一次データになる。
